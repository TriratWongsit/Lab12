# 📢 ข่าวจากผู้ใหญ่บ้าน (Village News)

## 📌 จุดประสงค์ของแอปพลิเคชัน
แอปพลิเคชันนี้ถูกพัฒนาขึ้นเพื่อสนับสนุนการสื่อสารภายในชุมชนแบบรอบด้าน ช่วยให้ผู้ใหญ่บ้านหรือผู้นำชุมชนสามารถกระจายข่าวสาร แจ้งเตือนภัยด่วน หรือนัดหมายกิจกรรมต่างๆ ถึงลูกบ้านได้อย่างรวดเร็วและทั่วถึง ช่วยลดปัญหาการตกหล่นของข้อมูลข่าวสาร และทำให้คนในชุมชนสามารถเตรียมพร้อมรับมือสถานการณ์ต่างๆ ได้ทันท่วงที

## 📝 รายละเอียดข้อมูลแอปพลิเคชัน
แอปพลิเคชันนี้พัฒนาด้วย **Flutter** โดยมีการออกแบบโครงสร้างและใช้งานเครื่องมือหลักตามที่กำหนดดังนี้:

* **ระบบจัดการสถานะ (State Management):** ใช้งานแพ็กเกจ `Provider` ในการจัดการสถานะ (State) ของรายการข่าวสาร เพื่อควบคุมการอัปเดตหน้าจอ UI แบบ Real-time เช่น เมื่อมีการเพิ่มข่าวใหม่ ค้นหาข่าว หรือกดบันทึกข่าว หน้าจอจะแสดงผลลัพธ์ทันที
* **ระบบฐานข้อมูล (Local Database):** ใช้งานแพ็กเกจ `sqflite` (SQLite) เป็นฐานข้อมูลภายในเครื่อง (Offline Database) เพื่อจัดเก็บข้อมูลข่าวสารทั้งหมดลงในมือถือ ทำให้ผู้ใช้งานสามารถเปิดอ่านข่าว หรือกด "บันทึกไว้อ่าน (Bookmark)" เพื่อกลับมาดูข้อมูลสำคัญย้อนหลังได้ แม้จะไม่มีการเชื่อมต่ออินเทอร์เน็ตก็ตาม
* **ฟีเจอร์หลัก (Features):**
  1. ระบบฟีดข่าวสาร แบ่งตามหมวดหมู่ (ทั่วไป, ด่วน, กิจกรรม)
  2. ระบบช่องค้นหาข่าวสาร (Search)
  3. ระบบเพิ่มข่าวสารใหม่ (ลง Local Database)
  4. ระบบลบข่าวสาร และบันทึกข่าวไว้อ่านออฟไลน์
